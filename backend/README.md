# Full Stack JS - Code Challenge (Backend)

`Code Challenge (Backend)` is the backend for the `Code Challenge (Frontend)` project.

## Features

-   Get List Files Endpoints.
-   Get File Data.

## Tech Stack

-   Javascript.
-   NodeJS.
-   ExpressJS.
